Instruções para Implantação do Site de Referência D&D no Netlify

Este documento contém instruções passo a passo para implantar o site de referência D&D no Netlify usando apenas um dispositivo móvel.

Nota: O Netlify é uma plataforma de hospedagem gratuita que permite implantar sites estáticos facilmente. Você não precisará de um computador para seguir estas instruções.

Passo 1: Criar uma conta no Netlify

  1. Abra o navegador do seu celular e acesse https://app.netlify.com/signup
  2. Você pode se cadastrar usando sua conta do GitHub, GitLab, Bitbucket ou e-mail
  3. Recomendamos usar a opção GitHub, pois facilitará os próximos passos
  4. Siga as instruções na tela para concluir o cadastro

Passo 2: Criar um repositório no GitHub

  1. Acesse https://github.com/new no navegador do seu celular
  2. Dê um nome ao repositório, como "dnd-reference-site"
  3. Deixe o repositório como público
  4. Marque a opção "Add a README file"
  5. Clique em "Create repository"

Passo 3: Fazer upload dos arquivos do site

  1. Descompacte o arquivo ZIP do site que você recebeu
  2. No seu repositório GitHub, clique no botão "Add file" e selecione "Upload files"
  3. Arraste ou selecione os arquivos da pasta "static" para upload
  4. Você precisará fazer isso várias vezes para cada pasta (css, js, data)
  5. Certifique-se de fazer upload do arquivo index.html e netlify.toml na raiz
  6. Após cada upload, adicione uma mensagem de commit como "Adicionar arquivos do site" e clique em "Commit changes"
Dica: Se estiver difícil fazer upload de muitos arquivos pelo celular, você pode usar o GitHub Desktop no computador ou pedir ajuda a alguém com acesso a um computador.

Passo 4: Implantar o site no Netlify

  1. Volte para o Netlify (app.netlify.com) e faça login
  2. Clique no botão "Add new site" e selecione "Import an existing project"
  3. Escolha "GitHub" como provedor de Git
  4. Autorize o Netlify a acessar seu GitHub, se solicitado
  5. Selecione o repositório "dnd-reference-site" que você criou
  6. Na seção de configurações de build:
    • Branch to deploy: main
    • Base directory: (deixe em branco)
    • Build command: (deixe em branco)
    • Publish directory: (deixe em branco, o netlify.toml já configura isso)
  7. Clique em "Deploy site"

Passo 5: Acessar seu site

  1. O Netlify começará a implantar seu site. Isso pode levar alguns minutos
  2. Quando a implantação for concluída, você verá um URL aleatório como "random-name-123456.netlify.app"
  3. Clique nesse URL para acessar seu site de referência D&D
  4. Você pode personalizar esse URL clicando em "Domain settings" e depois em "Options" > "Edit site name"

Solução de Problemas

Se o site não carregar corretamente:

  1. Verifique se todos os arquivos foram carregados corretamente no GitHub
  2. Certifique-se de que o arquivo netlify.toml está na raiz do repositório
  3. No Netlify, vá para a seção "Deploys" e clique no deploy mais recente para ver os logs
  4. Se necessário, clique em "Trigger deploy" para tentar novamente

Se os dados não aparecerem:

  1. Verifique se os arquivos JSON na pasta "data" foram carregados corretamente
  2. Abra o console do navegador (geralmente acessível nas opções de desenvolvedor) para ver mensagens de erro
Importante: Uma vez implantado, seu site estará disponível online e você poderá acessá-lo de qualquer dispositivo com acesso à internet. Não é necessário manter seu celular conectado para que o site funcione.
